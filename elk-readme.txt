ELK - Elasticsearch + Logstash + Kibana

Elasticsearch - nosql db based on lucene search engine which will store inputs/logs
Logstash - log piprpline tool that accepts inputs from various sources & exports data to various targets
Kibana - visualization UI layer which will help developers to monitor application logs

Logs (files) -> Logstash -> Elasticsearch -> Kibana
